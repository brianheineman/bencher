## `smtp`

이 섹션은 [SMTP](https://en.wikipedia.org/wiki/Simple_Mail_Transfer_Protocol) 서비스 구성을 지정합니다. 전체 섹션은 선택 사항입니다. 지정하지 않으면 모든 메시지가 대신 `logging`으로 전송됩니다.

|     이름      |            예시            | 기본값 | 필수 여부 |                                                  설명                                                   |
| :-----------: | :------------------------: | :----: | :-------: | :-----------------------------------------------------------------------------------------------------: |
|   hostname    |      "mailbonobo.com"      |  ---   |    예     |                                     SMTP 호스트 이름을 지정합니다.                                      |
|     port      |            587             |  587   |  아니오   |                                         SMTP 포트를 지정합니다.                                         |
| insecure_host |            ---             | false  |  아니오   |               SMTP 연결이 유효하지 않은 TLS 인증서를 허용할 수 있는지 여부를 제어합니다.                |
|   starttls    |            true            |  true  |  아니오   |                      SMTP 연결이 STARTTLS 프로토콜을 사용하는지 여부를 제어합니다.                      |
|   username    |         "bencher"          |  ---   |    예     |                            SMTP 호스트에서 사용할 사용자 이름을 지정합니다.                             |
|    secret     | "WM3F2u9cqSNdBPLfy9sJ5kk9" |  ---   |    예     | SMTP 호스트에서 사용할 사용자 이름의 비밀을 지정합니다. 로그에서 표시될 때 `************`로 숨겨집니다. |
|   from_name   |         "Bencher"          |  ---   |    예     |                          모든 이메일의 발신자 섹션에 나타날 이름을 지정합니다.                          |
|  from_email   | "info@bencher.example.com" |  ---   |    예     |                         모든 이메일의 발신자 섹션에 나타날 이메일을 지정합니다.                         |
