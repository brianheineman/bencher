## `smtp`

Этот раздел определяет конфигурацию службы [SMTP](https://en.wikipedia.org/wiki/Simple_Mail_Transfer_Protocol).
Весь раздел является необязательным.
Если не указано, все сообщения будут отправляться в `logging`.

|      Имя      |           Пример           | Значение по умолчанию | Обязательно |                                                    Описание                                                    |
| :-----------: | :------------------------: | :-------------------: | :---------: | :------------------------------------------------------------------------------------------------------------: |
|   hostname    |      "mailbonobo.com"      |          ---          |     Да      |                                           Указывает имя хоста SMTP.                                            |
|     port      |            587             |          587          |     Нет     |                                              Указывает порт SMTP.                                              |
| insecure_host |            ---             |         false         |     Нет     |                 Управляет разрешением недействительных TLS-сертификатов для подключения SMTP.                  |
|   starttls    |            true            |         true          |     Нет     |                       Управляет использованием протокола STARTTLS для подключения SMTP.                        |
|   username    |         "bencher"          |          ---          |     Да      |                                   Указывает имя пользователя на хосте SMTP.                                    |
|    secret     | "WM3F2u9cqSNdBPLfy9sJ5kk9" |          ---          |     Да      | Указывает секрет для имени пользователя на хосте SMTP. При записи он будет отображаться в виде `************`. |
|   from_name   |         "Bencher"          |          ---          |     Да      |                Указывает имя, которое будет отображаться в разделе "от" всех электронных писем.                |
|  from_email   | "info@bencher.example.com" |          ---          |     Да      |               Указывает email, который будет отображаться в разделе "от" всех электронных писем.               |
