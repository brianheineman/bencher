## `bencher logs`

Zeigen Sie die Ausgabe von den Bencher Self-Hosted Containern an,
wenn `bencher up` mit [dem `--detach`-Flag][detach flag] ausgeführt wurde.
Dies ist ähnlich dem Ausführen von [`docker compose logs`][docker compose logs].
