### `--console-volume <HOST_CONTAINER>`

Übergeben Sie ein Mount-Volume an den Console-Server-Container.
Das erwartete Format ist `/host/path:/container/path`.
Dies kann nützlich sein, wenn Sie mit [der Console Server-Konfiguration][console config] arbeiten.
Ähnlich zur [Option `--volume`][docker run volume] für `docker run`.

[docker run volume]: https://docs.docker.com/reference/cli/docker/container/run/#volume

[console config]: /de/docs/reference/console-config/
