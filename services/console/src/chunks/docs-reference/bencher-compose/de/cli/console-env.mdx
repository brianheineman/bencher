### `--console-env <KEY_VALUE>`

Übergebe eine Umgebungsvariable an den Console-Server-Container.
Das erwartete Format ist `KEY=Wert`.
Dies kann nützlich sein, wenn man mit [der Console Server-Konfiguration][console config] arbeitet.
Ähnlich wie die [Option `--env`][docker run env] für `docker run`.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[console config]: /de/docs/reference/console-config/
