### `--api-env <KEY_VALUE>`

Eine Umgebungsvariable an den API-Server-Container übergeben.
Das erwartete Format ist `SCHLÜSSEL=Wert`.
Dies kann nützlich sein, wenn Sie mit [der API-Server-Konfiguration][api config] arbeiten. Ähnlich wie die [die `--env`-Option][docker run env] für `docker run`.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[api config]: /de/docs/reference/server-config/
