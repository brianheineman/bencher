## `--api-volume <HOST_CONTAINER>`

Ein Volume-Mount an den API-Server-Container übergeben.
Das erwartete Format ist `/host/path:/container/path`.
Dies kann nützlich sein, wenn Sie mit [der API-Server-Konfiguration][api config] arbeiten.
Ähnlich wie die [Option `--volume`][docker run volume] bei `docker run`.

[docker run volume]: https://docs.docker.com/reference/cli/docker/container/run/#volume

[api config]: /de/docs/reference/server-config/
