## `bencher down`

Stoppen und entfernen der Bencher Self-Hosted Container,
wenn `bencher up` mit [dem `--detach`-Flag][detach flag] ausgeführt wurde.
Dies ist ähnlich wie das Ausführen von [`docker compose down`][docker compose down].
