## `bencher logs`

Afficher la sortie des conteneurs Bencher en auto-hébergement,
si `bencher up` a été exécuté avec [le drapeau `--detach`][detach flag].
Ceci est similaire à l'exécution de [`docker compose logs`][docker compose logs].
