### `--api-port <PORT>`

Spécifiez un numéro de port pour le conteneur du serveur API.
Par défaut, le port `61016` est utilisé.
Similaire à la forme de port uniquement de [l'option `--expose`][docker run expose] pour `docker run`.

[docker run expose]: https://docs.docker.com/reference/cli/docker/container/run/#publish
