### `--console-volume <HOST_CONTAINER>`

Passez un volume de montage au conteneur du serveur Console.
Le format attendu est `/host/path:/container/path`.
Cela peut être utile lors du travail avec [la configuration du serveur Console][console config].
Similaire à [l'option `--volume`][docker run volume] pour `docker run`.

[docker run volume]: https://docs.docker.com/reference/cli/docker/container/run/#volume

[console config]: /fr/docs/reference/console-config/
