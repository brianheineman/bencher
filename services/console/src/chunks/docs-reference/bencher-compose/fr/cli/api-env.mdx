### `--api-env <KEY_VALUE>`

Passez une variable d'environnement au conteneur du serveur API. Le format attendu est `KEY=value`. Cela peut être utile lors de l'utilisation de [la configuration du serveur API][api config]. Similaire à [l'option `--env`][docker run env] pour `docker run`.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[api config]: /fr/docs/reference/server-config/
