### `--console-port <PORT>`

Spécifiez un numéro de port pour le conteneur du serveur Console.
Par défaut, le port `3000` est utilisé.
Semblable à la forme uniquement port de [l'option `--expose`][docker run expose] pour `docker run`.

[docker run expose]: https://docs.docker.com/reference/cli/docker/container/run/#publish
