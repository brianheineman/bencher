### `--console-env <KEY_VALUE>`

Passez une variable d'environnement au conteneur du serveur Console.
Le format attendu est `KEY=value`.
Cela peut être utile lors de la configuration du [serveur Console][console config].
Similaire à l'option [`--env`][docker run env] pour `docker run`.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[console config]: /fr/docs/reference/console-config/
