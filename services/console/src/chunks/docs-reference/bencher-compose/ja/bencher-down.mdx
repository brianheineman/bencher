## `bencher down`

Bencherのセルフホスティングコンテナを停止し、削除します。ただし、[ `--detach` フラグ][detach flag]と共に`bencher up`を実行した場合に限ります。これは[`docker compose down`][docker compose down]を実行するのと類似しています。
