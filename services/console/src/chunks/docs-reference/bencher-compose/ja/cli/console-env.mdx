### `--console-env <KEY_VALUE>`

環境変数をコンソールサーバーコンテナに渡します。
期待されるフォーマットは `KEY=value` です。
[コンソールサーバーの設定][console config]で作業する際に役立ちます。
`docker run` の [ `--env` オプション][docker run env] と似ています。

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[console config]: /ja/docs/reference/console-config/
