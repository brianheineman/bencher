### `--console-volume <HOST_CONTAINER>`

マウントボリュームをコンソールサーバーコンテナに渡します。期待される形式は `/host/path:/container/path` です。これは、[コンソールサーバーの設定][console config] を操作する際に役立ちます。`docker run` の [`--volume` オプション][docker run volume] と似ています。

[docker run volume]: https://docs.docker.com/reference/cli/docker/container/run/#volume

[console config]: /ja/docs/reference/console-config/
