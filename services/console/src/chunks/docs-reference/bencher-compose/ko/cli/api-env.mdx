### `--api-env <KEY_VALUE>`

API 서버 컨테이너에 환경 변수를 전달합니다.
예상되는 형식은 `KEY=value`입니다.
이는 [API 서버 구성][api config] 작업 시 유용할 수 있습니다.
`docker run`의 [the `--env` option][docker run env]과 유사합니다.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[api config]: /ko/docs/reference/server-config/
