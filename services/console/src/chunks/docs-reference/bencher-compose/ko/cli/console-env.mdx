### `--console-env <KEY_VALUE>`

환경 변수를 Console 서버 컨테이너에 전달합니다. 예상 형식은 `KEY=value`입니다. 이는 [Console Server 구성][console config]과 작업할 때 유용할 수 있습니다. `docker run`의 [`--env` 옵션][docker run env]과 유사합니다.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[console config]: /ko/docs/reference/console-config/
