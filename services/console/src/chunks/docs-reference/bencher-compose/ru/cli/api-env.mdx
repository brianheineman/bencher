### `--api-env <KEY_VALUE>`

Передайте переменную окружения в контейнер сервера API.
Ожидаемый формат - `KEY=value`.
Это может быть полезно при работе с [конфигурацией API сервера][api config].
Аналогично [опции `--env`][docker run env] для `docker run`.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[api config]: /ru/docs/reference/server-config/
