### `--console-volume <HOST_CONTAINER>`

Передайте том монтирования в контейнер сервера Console.
Ожидаемый формат - `/host/path:/container/path`.
Это может быть полезно при работе с [конфигурацией сервера Console][console config].
Аналогично [опции `--volume`][docker run volume] для `docker run`.

[docker run volume]: https://docs.docker.com/reference/cli/docker/container/run/#volume

[console config]: /ru/docs/reference/console-config/
