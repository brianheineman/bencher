### `--console-env <KEY_VALUE>`

Передайте переменную окружения в контейнер сервера Console.
Ожидаемый формат — `KEY=value`.
Это может быть полезно при работе с [конфигурацией сервера Console][console config].
Аналогично [опции `--env`][docker run env] для `docker run`.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[console config]: /ru/docs/reference/console-config/
