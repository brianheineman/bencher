### `--console-env <KEY_VALUE>`

Pasa una variable de entorno al contenedor del servidor de la Consola.
El formato esperado es `LLAVE=valor`.
Esto puede ser útil cuando se trabaja con [la configuración del Servidor de Consola][console config].
Similar a [la opción `--env`][docker run env] para `docker run`.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[console config]: /es/docs/reference/console-config/
