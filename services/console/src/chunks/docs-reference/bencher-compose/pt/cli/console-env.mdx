### `--console-env <KEY_VALUE>`

Passe uma variável de ambiente para o contêiner do servidor Console.
O formato esperado é `KEY=value`.
Isso pode ser útil ao trabalhar com [a configuração do Servidor Console][console config].
Semelhante à [opção `--env`][docker run env] para `docker run`.

[docker run env]: https://docs.docker.com/reference/cli/docker/container/run/#env

[console config]: /pt/docs/reference/console-config/
