### `--console-volume <HOST_CONTAINER>`

Passe um volume de montagem para o contêiner do servidor Console.
O formato esperado é `/host/path:/container/path`.
Isso pode ser útil ao trabalhar com [a configuração do Servidor Console][console config].
Semelhante à [opção `--volume`][docker run volume] para `docker run`.

[docker run volume]: https://docs.docker.com/reference/cli/docker/container/run/#volume

[console config]: /pt/docs/reference/console-config/
