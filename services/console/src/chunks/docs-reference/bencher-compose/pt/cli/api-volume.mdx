## `--api-volume <HOST_CONTAINER>`

Transfira um volume de montagem para o contêiner do servidor API.
O formato esperado é `/host/path:/container/path`.
Isso pode ser útil ao trabalhar com [a configuração do Servidor API][api config].
Semelhante à [opção `--volume`][docker run volume] para `docker run`.

[docker run volume]: https://docs.docker.com/reference/cli/docker/container/run/#volume

[api config]: /pt/docs/reference/server-config/
