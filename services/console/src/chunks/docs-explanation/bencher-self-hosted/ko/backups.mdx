## 백업

Bencher 자체 호스팅 API 서버를 백업하는 가장 쉬운 방법은 [`bencher server backup` CLI 하위 명령어][bencher server backup]를 사용하는 것입니다. 이 하위 명령어를 사용하려면 사용자가 서버의 관리자인지 확인해야 합니다.
