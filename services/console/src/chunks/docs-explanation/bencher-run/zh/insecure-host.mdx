### `--insecure-host`

<br />

可选：允许与 Bencher API 服务器的不安全连接。

此标志在连接到使用自签名证书或不包含在系统证书存储中的证书的 Bencher 自托管实例时很有用。 它仅适用于 HTTPS 连接，因为 HTTP 连接本质上是不安全的。

**警告**：仅在经过验证的来源的安全网络中使用 `--insecure-host`，因为它会绕过 SSL 验证，可能使您暴露于[中间人攻击][wikipedia mitm]。
