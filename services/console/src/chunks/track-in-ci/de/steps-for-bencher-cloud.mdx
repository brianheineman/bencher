## Schritte für Bencher Cloud

1. [Erstellen Sie ein Bencher Cloud-Konto][bencher sign up].
2. [Erstellen Sie ein API-Token][console users tokens add] und fügen Sie es Ihrem CI als Geheimnis hinzu.
3. Erstellen Sie einen Workflow für Ihr CI, wie [GitHub Actions][github actions] oder [GitLab CI/CD][gitlab ci cd].
4. [Installieren Sie die Bencher CLI][install bencher cli] in Ihrem CI-Workflow.
