## Bencher Cloud 的步骤

1. [创建一个 Bencher Cloud 账户][bencher sign up]。
2. [创建一个 API 令牌][console users tokens add]，并将其作为机密添加到您的 CI 中。
3. 为您的 CI 创建一个工作流，例如 [GitHub Actions][github actions] 或 [GitLab CI/CD][gitlab ci cd]。
4. 在您的 CI 工作流中[安装 Bencher CLI][install bencher cli]。
