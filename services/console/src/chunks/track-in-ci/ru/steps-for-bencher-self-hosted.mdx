## Шаги для Bencher Self-Hosted

1. Создайте [экземпляр Bencher Self-Hosted][bencher self hosted].
2. Зарегистрируйте аккаунт на вашем экземпляре Bencher Self-Hosted.
3. Создайте API токен и добавьте его в CI как секрет.
4. Создайте рабочий процесс для вашего CI, например [GitHub Actions][github actions] или [GitLab CI/CD][gitlab ci cd].
5. [Установите Bencher CLI][install bencher cli] в ваш CI рабочий процесс.
   Убедитесь, что версия CLI соответствует версии вашего экземпляра Bencher Self-Hosted.
6. {props.step}
