Бинарные размеры могут увеличиваться со временем, особенно в {props.lang}. [Континуальное бенчмаркинг][continuous benchmarking] — это практика запуска бенчмарков для каждого изменения, чтобы убедиться, что изменения не приводят к ухудшению производительности. Вы можете рассматривать размеры бинарных файлов как бенчмарк и отслеживать их в CI, чтобы обнаружить слишком большие размеры бинарных файлов. Самый простой способ реализовать континуальное бенчмаркинг для размеров бинарных файлов {props.lang} — использовать Bencher.

[continuous benchmarking]: /ru/docs/explanation/continuous-benchmarking/
