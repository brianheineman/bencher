## Bencher Cloud の手順

1. [Bencher Cloud アカウントを作成する][bencher sign up]。
2. [API トークンを作成][console users tokens add]し、それを秘密として CI に追加する。
3. [GitHub Actions][github actions] や [GitLab CI/CD][gitlab ci cd] のような CI のワークフローを作成する。
4. [Bencher CLI をインストール][install bencher cli]して CI ワークフローに組み込む。
