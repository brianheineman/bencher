## Passos para o Bencher Cloud

1. [Crie uma conta no Bencher Cloud][bencher sign up].
2. [Crie um token de API][console users tokens add] e adicione-o ao seu CI como um segredo.
3. Crie um fluxo de trabalho para o seu CI, como [GitHub Actions][github actions] ou [GitLab CI/CD][gitlab ci cd].
4. [Instale o Bencher CLI][install bencher cli] no seu fluxo de trabalho CI.
