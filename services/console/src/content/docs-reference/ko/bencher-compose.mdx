[Bencher Self-Hosted][bencher self hosted]를 배포하는 가장 쉬운 방법은 `bencher` CLI 하위 명령어 세 가지를 사용하는 것입니다:

- [`bencher up`][bencher up]: Bencher Self-Hosted 컨테이너를 실행합니다.
  이것은 [`docker compose up`][docker compose up]을 실행하는 것과 유사합니다.
- [`bencher logs`][bencher logs]: `bencher up`이 [the `--detach` flag][detach flag]와 함께 실행되었다면 Bencher Self-Hosted 컨테이너의 출력을 봅니다.
  이것은 [`docker compose logs`][docker compose logs]를 실행하는 것과 유사합니다.
- [`bencher down`][bencher down]: `bencher up`이 [the `--detach` flag][detach flag]와 함께 실행되었다면 Bencher Self-Hosted 컨테이너를 중지하고 제거합니다.
  이것은 [`docker compose down`][docker compose down]을 실행하는 것과 유사합니다.
