La manera más fácil de desplegar [Bencher Self-Hosted][bencher self hosted] es usando un tríptico de subcomandos de la CLI `bencher`:

- [`bencher up`][bencher up]: Levantar los contenedores de Bencher Self-Hosted.
  Esto es similar a ejecutar [`docker compose up`][docker compose up].
- [`bencher logs`][bencher logs]: Ver la salida de los contenedores de Bencher Self-Hosted,
  si `bencher up` fue ejecutado con [el flag `--detach`][detach flag].
  Esto es similar a ejecutar [`docker compose logs`][docker compose logs].
- [`bencher down`][bencher down]: Detener y eliminar los contenedores de Bencher Self-Hosted,
  si `bencher up` fue ejecutado con [el flag `--detach`][detach flag].
  Esto es similar a ejecutar [`docker compose down`][docker compose down].
